# 🛍️ ShopVerse - Dynamic React Shopping Website

ShopVerse is a modern and colorful e-commerce website built using React and React Router. The project features a responsive user interface, smooth animations, product listings, shopping cart functionality, and multiple pages for a complete shopping experience.

## 🚀 Features

- 🎨 Modern and colorful UI/UX
- ✨ Animated gradient backgrounds and hover effects
- 🛒 Add-to-Cart functionality
- 📦 Orders page
- 📞 Contact page with headquarters and customer support details
- 🏠 Home page with hero section and featured products
- ℹ️ About page
- 🔗 React Router navigation
- 📱 Responsive design for desktop and mobile devices
- 🧩 Reusable React components
- ⚡ Built with Vite for fast performance

## 🛠️ Technologies Used

- React.js
- React Router DOM
- JavaScript (ES6+)
- CSS3
- Vite

## 🌟 Pages

### Home

Displays the main landing page with:

* Header
* Navigation Bar
* Hero Section
* Product Showcase
* Footer

### About

Provides information about ShopVerse and its mission.

### Contact

Displays ShopVerse headquarters location, customer support helpline, email support, and working hours.

### Cart

Shows products added by the user through the Add To Cart button.

### Orders

Displays order-related information and future order history.

## ⚙️ Installation

1. Clone the repository

```bash
git clone <repository-url>
```

2. Navigate to the project folder

```bash
cd shopverse
```

3. Install dependencies

```bash
npm install
```

4. Start the development server

```bash
npm run dev
```

5. Open your browser and visit

```text
http://localhost:5173
```
